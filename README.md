# Love Letters to Bash, Git, and Github
## Introduction
These are a small collection of letters written to Bash, Git, and Github, in an assignment design to reinforce the basic commands in a fun and interesting way! I hope you enjoy reading them!

* [GITHUB](https://github.com/)
